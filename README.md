# Online Banking Automation Framework

## Project Description
Automation framework for an online banking web application using Selenium WebDriver and Java.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

## Automated Modules
- Login & Logout
- Account Summary
- Fund Transfer
- Transaction History

## How to Run
1. Clone the repository
2. Import as Maven project
3. Run test cases using TestNG
